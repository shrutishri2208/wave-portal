## Cloning and running the application locally
### Clone the project 
```
git clone https://github.com/shrutishri2208/wave-portal.git
```

```
cd wave-portal

```

### Install all the dependencies
```
npm install
npm install ethers
```

### Run the application
```
npm start
```
